# 项目静态资源目录 (Project Assets)

为了提升 README 的视觉吸引力，建议在该目录下上传以下图片，并在 `README.md` 中引用：

### 建议上传的文件清单 (Suggested Files):

1. **architecture.png**: 系统架构图，展示多智能体编排流程、SSE 通信机制以及前后端交互逻辑。
2. **screenshot-home.png**: 项目首页截图，展示简洁现代的 UI 设计。
3. **screenshot-editor.png**: 文章生成过程中的实时打字机效果截图（SSE 效果展示）。
4. **screenshot-admin.png**: 后台管理与统计报表页面截图。
5. **demo.gif**: (可选) 录制一段 15-30 秒的完整创作流程动图，这是最能直观展示项目价值的方式。

### 如何在 README 中引用 (How to reference):

```markdown
![系统架构](./docs/images/architecture.png)
```

> **注意**: 保持图片比例一致，建议使用 WebP 或 PNG 格式以平衡清晰度与加载速度。
